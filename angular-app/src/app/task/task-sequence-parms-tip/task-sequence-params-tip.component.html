Explanation of Parameters

Population Size: Determines the number of task sequences selected to form a new generation.

Number of Generations: Specifies the total number of generations to be executed.

Crossover Probability: Represents the likelihood of crossover occurring.

Mutation Probability: Indicates the likelihood of mutation taking place.

Elitism: Sets the number of top individuals to be replicated in the next generation.

Stabilization: Defines the number of generations executed without improvement in the best individual.

Ideal Cost: Sets the target cost that the algorithm aims to achieve.

You can review all generations in the log file.
